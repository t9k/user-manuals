# 辅助

TensorStack AI 平台的辅助模块汇集了辅助性的资源，它们为其他资源提供支持，或者为用户提供方便、实用的功能。总结如下：

|              | 来源 | 作用                              |
| ------------ | --------------- | --------------------------------- |
| Secret       | Kubernetes  | 存储敏感信息并用于配置 |
| ConfigMap    | Kubernetes  | 存储非敏感信息并用于配置  |
| Pod          | Kubernetes  | 运行容器，测试镜像和应用程序  |
| ImageBuilder | TensorStack | 构建自定义镜像    |
