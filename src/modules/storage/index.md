# 存储

Storage（存储）模块汇集了存储系统相关的资源，列表如下：

|              | 来源 | 作用                              |
| ------------ | --------------- | --------------------------------- |
| PVC          | Kubernetes              | 动态申请集群持久存储                          |
| StorageShim  | TensorStack              | 对各种存储系统的支持 |
| Explorer     | TensorStack              | 文件浏览器，查看和管理 PVC 中的文件       |
